## Getting Started
To set up the project environment, please follow these steps:

1. **Clone Repositories:** Clone `guimottifx`, `libmottifx`, and `procmottifx` into the same parent directory.
2. **Build Initialization:** Once the folders are placed correctly, you **must** run the build script to prepare the environment.
   * If you use Make: Run `make` in your terminal.
   * If you use the shell script: Run `./mottifx.sh`.
3. **Run:** After the initialization is complete, you can launch the project.
